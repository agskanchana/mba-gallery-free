# Changelog

All notable changes to Medical Before After Gallery will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.4] - 2024-08-02

### Added
- Comprehensive color theming system with CSS custom properties
- Gradient support for color themes
- Enhanced admin color picker with presets
- Improved mobile filter toggle button styling
- Better accessibility features throughout the interface
- Enhanced error logging and debugging capabilities

### Fixed
- JavaScript scope error with `closeSidebarOnMobile` function
- Modal navigation issues on mobile devices
- Filter clearing functionality improvements
- Drag performance on before/after slider
- Mobile responsiveness issues

### Changed
- Optimized image loading performance
- Enhanced Freemius integration
- Improved plugin security with nonce verification
- Better internationalization support
- Updated plugin header information

### Security
- Added enhanced nonce verification for AJAX calls
- Improved sanitization of user inputs
- Better error handling to prevent information disclosure
